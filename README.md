# RL Chess AI

This is an experimental Chess implementation which may use different strategies: Reinforcement learning, MCTS (Monte Carlo Tree Search), MiniMax with Alpha/Beta-Pruning.

The implementation is completely based on HTML, CSS and JavaScript. It consists of one single HTML-file (that also includes .css styles and JavaScript code).


You may use it to play chess yourself (you vs. computer) or to watch chess games (computer vs. computer). You may change the strategy being used (see above). You may extend or change the code.

# Running the Game

Just open the HTML-file  with a browser.
